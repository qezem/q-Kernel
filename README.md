# Installation

## Ensure that you have the following dependencies:
- i386-elf-gcc
- i386-elf-ld
- i386-elf-objcopy
- nasm
- qemu-full

## 1. Clone the repository
```bash
git clone "https://github.com/qezem/q-Kernel.git"
cd q-Kernel
```
## 2. Build the kernel
```bash
make
```
## 3. Run kernel
```bash
make run
```
